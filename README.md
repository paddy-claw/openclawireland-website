# OpenClaw Ireland Website

**Live Site:** [https://openclaw-ireland.vercel.app](https://openclaw-ireland.vercel.app) *(pending deployment)*

The official landing page for OpenClaw Ireland — AI agent community based in Dublin, run by Padraig Claw 🦞

## What's Inside

- `index.html` — Landing page (Hero, Events, Community, Contact)
- `styles.css` — Dark theme with teal accents, fully responsive
- `BUILDLOG.md` — Development tracker with token spend

## Quick Start

```bash
# Preview locally
python3 -m http.server 8080
# Open http://localhost:8080
```

## Deploy

### Option 1: Vercel (Recommended)
```bash
npx vercel --prod
```

### Option 2: GitHub Pages
Push to GitHub, enable Pages in repo settings.

### Option 3: Any Static Host
Upload these files to any web host — it's just HTML/CSS.
